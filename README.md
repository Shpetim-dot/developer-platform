# Developer Platform API 🚀  ## Beschreibung  Die Developer Platform ist eine Backend-API, die entwickelt wurde, um Projekte zu verwalten. Die API kann Projektdaten empfangen, speichern, aktualisieren und löschen.  Das Projekt wurde mit Python und FastAPI erstellt und zeigt grundlegende Kenntnisse in: - REST APIs - Backend-Entwicklung - Datenverarbeitung mit JSON - Git & GitHub - Softwarestrukturierung   ## Features  ✅ Projekte anzeigen   ✅ Einzelne Projekte suchen   ✅ Neue Projekte erstellen   ✅ Projekte aktualisieren   ✅ Projekte löschen     ## Technologien  | Technologie | Verwendung | |---|---| | Python | Programmiersprache | | FastAPI | Erstellung der REST API | | Pydantic | Datenvalidierung | | JSON | Datenaustausch | | Git/GitHub | Versionskontrolle |   ## API Endpunkte  ### Startseite
